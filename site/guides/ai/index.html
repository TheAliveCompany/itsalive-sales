AI Integration Guide

Add Claude, GPT, and Gemini to your app

itsalive.co provides unified access to the best AI models through a simple API. Chat with AI, generate images, transcribe audio, and synthesize speech - all billed through your itsalive credits.

How Credits Work

AI features use prepaid credits. 1 credit = $0.001 (1/10 of a cent). Purchase credits in your dashboard and they're shared across all your apps.

Available Models

We offer two tiers for each provider:

  • Good - Fast, cost-effective models for most tasks
  • Best - Most capable models for complex reasoning

Basic Usage

// Simple chat request
const response = await fetch('/_ai/chat', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  credentials: 'include',
  body: JSON.stringify({
    provider: 'claude',  // 'claude', 'gpt', or 'gemini'
    tier: 'good',        // 'good' or 'best'
    messages: [
      { role: 'user', content: 'What is the capital of France?' }
    ]
  })
});

const { content, usage } = await response.json();
console.log(content);  // "The capital of France is Paris."
console.log(usage.credits_used);  // 1

Image Analysis (Vision)

All chat models support image input:

const response = await fetch('/_ai/chat', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  credentials: 'include',
  body: JSON.stringify({
    provider: 'claude',
    tier: 'good',
    messages: [{
      role: 'user',
      content: [
        { type: 'text', text: 'What is in this image?' },
        { type: 'image', url: 'https://example.com/photo.jpg' }
      ]
    }]
  })
});

Speech-to-Text (Whisper)

Transcribe audio files. Costs ~10 credits per minute of audio.

const formData = new FormData();
formData.append('file', audioFile);

const response = await fetch('/_ai/transcribe', {
  method: 'POST',
  credentials: 'include',
  body: formData
});

const { text, duration, credits_used } = await response.json();
console.log(text);  // Transcribed text

Text-to-Speech

Generate natural-sounding speech. Costs ~20 credits per 1,000 characters (40 for HD).

Available voices: alloy, echo, fable, onyx, nova, shimmer

const response = await fetch('/_ai/tts', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  credentials: 'include',
  body: JSON.stringify({
    input: 'Welcome to our application!',
    voice: 'nova',
    model: 'tts-1'  // or 'tts-1-hd' for higher quality
  })
});

// Response is audio binary
const audioBlob = await response.blob();
const audioUrl = URL.createObjectURL(audioBlob);
const audio = new Audio(audioUrl);
audio.play();

Error Handling

const response = await fetch('/_ai/chat', { ... });

if (response.status === 402) {
  const { balance, required } = await response.json();
  alert(`Insufficient credits. You have ${balance}, need ${required}.`);
} else if (response.status === 401) {
  alert('Please log in to use AI features.');
} else if (!response.ok) {
  const error = await response.json();
  console.error('AI error:', error.details);
}

Tips

  • Start with Good tier - It handles most tasks well and costs less
  • Use Best tier for quality-critical tasks - Complex reasoning, important content
  • GPT-5 Mini for volume - When you need thousands of requests cheaply
  • Claude for coding - Consistently best at code generation and review
  • Set system prompts - Guide the model's behavior for consistent results
  • Monitor usage - Check /_ai/usage to track spending
